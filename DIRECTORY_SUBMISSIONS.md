## Submission Best Practices

1. **Use consistent information** across all directories
2. **Write unique descriptions** for each directory (avoid duplicate content)
3. **Include relevant keywords** naturally in descriptions
4. **Add high-quality images** where allowed
5. **Verify listings** via email/phone when required
6. **Monitor and respond** to any reviews or questions
7. **Update listings** if information changes
